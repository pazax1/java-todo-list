# Java To Do List

A simple console-based To Do List application written in Java.

This project was created to practice:
- Object-Oriented Programming (OOP)
- Working with classes and methods
- Exception handling
- ArrayList collections
- Separating application responsibilities into different classes

## Features

- Add new tasks
- Remove tasks
- Display task list
- Validate user input
- Handle incorrect user input using exceptions

## Technologies

- Java
- Scanner
- ArrayList
- Exception Handling
